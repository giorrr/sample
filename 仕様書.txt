問題１，２に共通する仕様変更

変更した仕様書
・セルに含まれる文字はASCII印字可能文字です。(0x20-0x7e)

→セルに含まれる文字はASCII印字可能文字です。(0x20-0x7e)
　ただし、グループ化以外の用途で半角コロン(:)を使用しないこと
　理由：グループ化のコロンなのか値としてのコロンなのか判別できないため

動作環境　Windows
使用言語　Python3



問題１　第一正規化の仕様書

コンパイルするためのコマンドライン引数について
実行するファイルが存在するディレクトリで

python 1_reg.py 処理するファイル名 処理するファイル名のヘッダー有無

と入力

コマンドライン入力例(Windows)
data.tsvを処理したい場合
python 1_reg.py data.tsv 1
python 1_reg.py ./data.tsv 1
python 1_reg.py C:\Users\aa\Desktop\data.tsv 1

-処理するファイル名は相対パスでも絶対パス表記でも良い
-ヘッダーの有無は半角の1ならばヘッダーが存在し、半角の0ならばヘッダーが存在しないファイルとして処理される
-出力されるファイルは1reg.tsvで、実行するファイル(1_reg.py)が存在するディレクトリに出力される。
